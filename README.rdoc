== README

A rails app with blog posts and comments.

http://localhost:3000/ - displays /posts

http://localhost:3000/posts - lists all post titles and codies in an unordered list.

http://localhost:3000/comments - lists all comments in an ordered list.

http://localhost:3000/posts/new - has a form for creating a new blog post. fields for title and body, with a button to submit and save the post.
